# -
Симуляция, показывающая поверхностное натяжение воды

Зажать ПКМ — перемещать камеру
Нажать на красную коробку — наполнить стакан
Можно перестаскивать монетки нажатием ЛКМ
При опускании монетки в стакан, уровень воды увеличивается
При превышении водой горлышка, появляется обозначение радиуса окружности (а также верхний переменный радиус эллипса), часть дуги которой это как раз натяжение воды.
